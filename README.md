# Student Mark Analytics Project

A Python-based Student Mark Report System that manages student academic records using **Pandas** and **CSV** storage. Built with a clean modular structure where each feature is separated into its own file.

---

## Features

1. Display all students with marks, total, average, result, and cutoff scores
2. Search student by Register Number
3. Add new student with mark validation and duplicate register number check
4. Statistics Menu:
   - Total students, pass and fail count
   - Number of centum scorers in a subject
   - Average mark in a subject
   - Maximum mark in a subject (with student name)
   - Minimum mark in a subject (with student name)
   - List of students failed in a subject
5. Export Menu:
   - Top 10 students by total marks
   - Failed students list
   - Subject-wise summary report

---

## Project Structure

```
StudentMarkReport/
│
├── data/
│   ├── Studentdata.csv        # Main student records (125 students)
│   ├── top10_students.csv     # Generated by export module
│   └── failed_students.csv    # Generated by export module
│
├── modules/
│   ├── __init__.py            # Makes modules folder a Python package
│   ├── load_data.py           # Reads and saves CSV - used by all modules
│   ├── display.py             # Display all students and search by register no
│   ├── add_student.py         # Add new student with validation
│   ├── statistics.py          # All statistical operations
│   └── export.py              # Export reports to CSV
│
├── Main.py                    # Menu-driven program connecting all modules
├── Generate csv.py            # One-time script to generate Studentdata.csv
└── README.md
```

---

## Technologies Used

- Python 3.12
- Pandas (CSV read/write and data analysis)
- CSV module (data generation)
- OS module (dynamic file path handling)

---

## How to Run

**Step 1 - Install pandas:**
```
pip install pandas
```

**Step 2 - Generate CSV (run only once):**
```
python Generate csv.py
```

**Step 3 - Run the project:**
```
python Main.py
```

---

## Key Concepts Used

- Modular programming (each feature in separate file)
- Pandas DataFrame operations (read_csv, to_csv, sort_values, apply, iterrows)
- Input validation (marks between 0-100, duplicate register number check)
- CSV as persistent data storage
- Dynamic file path handling using os.path

---

## Dataset

- 125 student records
- 6 subjects: Tamil, English, Maths, Physics, Chemistry, Biology
- Pass mark: 35 per subject
- Engineering cutoff = Maths + (Physics + Chemistry) / 2
- Medical cutoff = (Physics + Chemistry) / 2 + Biology
